Мониторинг потребностей жителей города в развитии инфраструктуры

Картографический сервис программы мониторинга потребностей жителей в развитии инфраструктуры городской среды на разных уровнях централизации, на основе экспериментального изучения их ценностно-смысловой сферы.

О системе

На сайте представлено 2 сервиса, использующие данные из социальных сетей.

1. Cервис мониторинга потребностей жителей в развитии инфраструктуры городской среды.

2. Сервис мониторинга происшествий на основе данных из социальных сетей.


1. Мониторинг потребностей жителей в развитии инфраструктуры

В сервисе использован метод мониторинга потребностей жителей в развитии инфраструктуры городской среды, на основе ценностно-смысловых аспектов городского пространства по данным из социальных сетей. В ходе реализации проекта, были изучены взаимосвязи ценностно-смысловых установок жителей Санкт-Петербурга, их потребностей в развитии инфраструктур и компонентов их профилей в социальной сети Вконтакте для дальнейшего прогнозирования потребностей в изменении городской среды с опорой на анализ социальных профилей, поиск активных жителей города, готовых вступать в диалог с управляющими организациями и включаться в проекты соучастного проектирования.


В результате исследования были выделены основные группы объектов инфраструктуры, в которых нуждаются жители города. На карте представлена визуализация субъективной структуры многомерного пространства потребностей жителей города в развитии инфраструктуры по районам. Для каждой группы объектов инфраструктуры было определено эвклидово расстояние в многомерном пространстве потребностей и рассчитан индекс значимости данных объектов для каждого района. На карте представлено отклонение от среднего значения индекса значимости по каждому району. Чем меньше индекс, тем больше в данном районе жителей нуждаются в развитии данной инфраструктуры.

2. Мониторинг происшествий по данным из социальных сетей
Источник данных о происшествиях: информация, полученная из социальных сетей. В качестве источника информации выбраны крупные сообществв ВКонтакте о ДТП и ЧП.

Для определения адреса и тематики, используется модуль по выделению адреса и типа происшествия из текстовых данных социальных сетей. В качестве тем инцидентов для распознавания были выделены следующие: Угон, Авария, Пожар.
